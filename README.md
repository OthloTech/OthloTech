# OthloTech

[OthloTech](www.othlo.tech)の公式サイトです。

### セットアップ

```
npm install
```

### ビルド

開発用ビルド + 変更の監視

```
npm run gulp
# => localhost:8888 が立ち上がります
```

### 技術スタック
